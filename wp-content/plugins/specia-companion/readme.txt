== Description ==

Specia Companion plugin to enhance the functionality of free themes made by Specia Themes.


== Installation ==

= Option 1. =

- Upload the entire `specia-companion` zip file to the `/wp-content/plugins/` directory.
- Activate the plugin through the 'Plugins' menu in WordPress.

= Option 2. =

- From your wp-admin —> Plugins —> Add New —> Search "Specia Companion" —> Click install.
- After plugin is downloaded, click active plugin



== Frequently Asked Questions ==

= How to Install Plugin ? =
Just upload the `specia-companion.zip` to the `/wp-content/plugins/` directory, and then activate it from 'Plugins' menu in WordPress.

= This plugin only work for Specia themes ? =
Yes, This plugin is specially make for Specia Themes . You can install theme for better compatibility.
